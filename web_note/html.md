
# HTML

## 1.HTML描述  
HyperText Markup Language

标签(Markup Tag)：用< >包裹的具有一定含义的内容，如：<strong>  </storng>

HTML不是编程语言，没有逻辑。只是一种标记语言。

## 2.HTML结构
```
<!DOCTYPE html>  //html5解析
<html>
<head>
<meta charset="utf-8"> <!--编码方式-->
<title>标题</title>

</head> <!头部，网页相关设置，不显示>
<body>
<p>段落</p>
<!脚本>
<script> 


</script>


</body>
</html>
```
```

<div></div> 用来布局的，无含义
```
